## IPアドレスとは

インターネット上のすべての機器（PC・スマホ・サーバー）が持つ「住所」。
データを届けるために必要。

---

## IPv4アドレス

```
例：192.168.1.10
```

- `0〜255` の数字4つをドット(`.`)でつないだ形式
- 全世界で約 **43億個** 存在
- アドレス枯渇問題 → IPv6が登場（後述）

---

## グローバルIPとプライベートIP

```
インターネット
    │
  ルーター ← グローバルIP（例：203.0.113.5）※世界で唯一
    │
  LAN（家庭・社内ネットワーク）
  ├── PC       ← プライベートIP（例：192.168.1.2）
  ├── スマホ   ← プライベートIP（例：192.168.1.3）
  └── テレビ  ← プライベートIP（例：192.168.1.4）
```

| 種類 | 用途 | 例 |
|---|---|---|
| **グローバルIP** | インターネット上の住所。世界に1つだけ | `203.0.113.5` |
| **プライベートIP** | 家庭・社内ネットワーク内の住所 | `192.168.x.x` |

### プライベートIPの範囲

```
10.0.0.0     〜 10.255.255.255
172.16.0.0   〜 172.31.255.255
192.168.0.0  〜 192.168.255.255
```

> [!tip] ポイント
> 家のWi-Fi内の機器はそれぞれ別のプライベートIPを持つが、
> インターネット側からはルーターのグローバルIPが1つに見える。
> この変換の仕組みが **NAT** → [[IT学習/フェーズ1_ネットワーク基礎/02_NAT・ポート番号|次回学習]]

## サブネットマスクとCIDR

IPアドレスには必ず **サブネットマスク** がセットになっている。

```
IPアドレス    ：192.168.1.10
サブネットマスク：255.255.255.0
```

### CIDR（サイダー）表記

```
192.168.1.10/24
```

`/24` = 先頭から **24ビット** がネットワーク部

```
192.168.1.  10
└─────────┘ └─┘
ネットワーク部  ホスト部
```

### よく使うCIDRとホスト数

| CIDR | サブネットマスク | 使えるホスト数 |
|---|---|---|
| /24 | 255.255.255.0 | 254台 |
| /16 | 255.255.0.0 | 65,534台 |
| /8 | 255.0.0.0 | 約1,677万台 |

## 確認テスト

| 問 | 問題 | 答え | 結果 |
|---|---|---|---|
| 1 | `192.168.0.50` はグローバル？プライベート？ | プライベートIP | ✅ |
| 2 | 同じWi-Fi内のPC・スマホ、インターネット側から見たIPは？ | 同じ（ルーターのグローバルIP） | ✅ |
| 3 | `10.0.0.1/24` の `/24` は何を表す？ | 先頭24ビットがネットワーク部 | ✅ |
